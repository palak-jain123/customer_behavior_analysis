Project overview 
The Customer Behavioral Dashboard is a data analytics project designed to analyze customer purchasing behavior and generate actionable business insights.
The dashboard helps businesses understand customer trends, spending patterns, purchase frequency, and customer segments for better decision-making and targeted marketing strategies.

This project combines data analysis, visualization, and business intelligence techniques to transform raw customer data into meaningful insights.

🎯 Objectives
Analyze customer purchasing behavior
Identify high-value and low-value customers
Track sales trends and customer engagement
Perform customer segmentation analysis
Create interactive visualizations for business insights
📂 Dataset Information

The dataset contains customer transaction and purchasing details such as:

Customer ID
Gender
Age
Product Category
Purchase Amount
Purchase Frequency
Payment Method
Region
Ratings / Reviews
Order Date
